# Thai Meet

Thai Meet is a Thailand-focused Android/iOS dating app concept and MVP plan.

The current repository contains the product requirements, benchmark material, and planning artifacts for the initial Flutter mobile app and NestJS backend build.

## Repository Layout

```text
docs/
  prd/        Product requirements documents
  designs/    Office-hours and planning review artifacts
benchmark/    Competitor/reference material
```

Planned implementation layout:

```text
apps/
  mobile/     Flutter Android/iOS app
  api/        NestJS modular monolith
  admin/      Admin web app
packages/
  api-contracts/
infra/
```
