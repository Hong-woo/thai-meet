Interaction Contract

Press

Scale to 0.97 for 140ms on buttons, chips, nav, and icon taps.

Swipe

Gesture cards should use interruptible spring motion; no keyframe restart.

Contact Card

Reveal with 220ms ease-out; never animate raw contact text into chat.

Sheet

Open from trigger context, use short ease-out, and keep handle visible.

Accessibility Contract

Focus

Keyboard focus uses a visible rose ring; dark swipe screens use a pale ring.

Current Tab

Active bottom nav items expose `aria-current="page"`.

Progress

Swipe stack progress uses `role="progressbar"` with numeric values.

Motion

Reduced-motion users keep layout changes without visible animation.
